Overview

  • What is “tidy” data? (Next slide)
  • Tidyverse = “universe” of add-on libraries for R, which assume tidy data so as to streamline data cleaning/analysis!

Tidy Data

  • Every row represents an observation
  • Every column represents an attribute
  • Every cell contains a value \(V_{{\color{#E69F00}i}{\color{#56B4E9}j}}\) = attribute \(j\) for observation \(i\)
✅ Tidy!
id name flights
DCA Reagan National 11.55
BWI Baltimore-Wash Int'l 11.15
IAD Dulles International 10.27

Tidy Data Check

  1. Choose a row, take note of its ID
  2. Go through values in row, asking yourself: is this the value of some attribute of [observation]?
❌ Not Tidy!
id var value
DCA name Reagan National
DCA flights 11.55
BWI name Baltimore-Wash Int'l
BWI flights 11.15
IAD name Dulles International
IAD flights 10.27

Above: 11.55 is the value of an attribute of DCA (number of flights in 2022, in millions); Right: name is not a property of DCA

Factors for Ordering Plot Elements

relig_summary <- gss_cat %>%
  group_by(relig) %>%
  summarise(
    age = mean(age, na.rm = TRUE),
    tvhours = mean(tvhours, na.rm = TRUE),
    n = n()
  )
relig_labs <- labs(
  x = "TV Hours / Day",
  y = "Religion"
)

Without forcats:

Code 
relig_summary |>
  ggplot(aes(tvhours, relig)) +

Code 
relig_summary |>
  mutate(relig = fct_reorder(relig, tvhours)) |>

    geom_segment(aes(yend = relig, x=0, xend = tvhours)) +
